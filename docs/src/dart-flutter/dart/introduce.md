# Dart 初体验

## 获取 Dart SDK

- 根据官网的介绍和提示，进行选择[获取 Dart SDK](https://dart.cn/get-dart)
- 直接下载: [Windows 下载](https://gekorm.com/dart-windows/)

## 在 vs code 中安装插件

- Dart
  ![dart](/image/dart.png)
- code Runner
  ![code runner](/image/codeRunner.png)

## 创建 Dart 文件

Dart 文件的扩展名为`.dart`

- `hello.dart`

所有的代码都要写在`main`函数中

```dart
void main() {
  print('Hello Dart!');
}
```

- 右键后选择 `run code`运行文件
  ![Alt text](/image/run.png)

- 在控制台得到结果
  ![Alt text](/image/dartResult.png)
