# 目录结构

```bash

- blog -|-- public
        |
        |-- scaffolds
        |            |-- _posts
        |            |
        |-- source --|-- about
        |            |
        |            |-- ...
        |
        |-- theme
        |
        |-- _config.yml
        |
        |-- _config.landscape.yml
```

- public: 这个文件夹用于存放打包后的文件
- source: 这个文件夹为主要的文件夹
  - \_posts: 文章存放的文件夹
  - about: 关于页面的内容
  - ...
- theme: 这个文件夹用于存放主题
- \_config.yml: 项目的配置文件
- ...
